# Summer Hiking Series 2026

A weekend-by-weekend day hiking plan from the first day of spring to the last day of fall, focused on trails in and around the Seattle metro area.

## Overview

- **Duration:** March 21 - December 19, 2026 (40 weekends)
- **Seasons:** Spring (13 weekends), Summer (14 weekends), Fall (13 weekends)
- **Focus:** Day hikes only, no backpacking or multi-day treks
- **Region:** Seattle metro area, with select "worth the drive" destinations

## Project Structure

```
2026/
  data/
    master_schedule.csv      # All 40 weekends with hike assignments
    spring_hikes.csv         # Detailed spring trail data
    summer_hikes.csv         # Detailed summer trail data
    fall_hikes.csv           # Detailed fall trail data
  spring/
    guide.md                 # Spring hiking guide and recommendations
  summer/
    guide.md                 # Summer hiking guide and recommendations
  fall/
    guide.md                 # Fall hiking guide and recommendations
```

## How to Use

Start with `2026/data/master_schedule.csv` for a week-by-week overview. Each seasonal guide in `2026/{season}/guide.md` has details on trail highlights, best times, and what to expect. The CSV files in `2026/data/` have structured trail data for sorting and filtering.

## Passes and Permits

- **Discover Pass:** $35/year or $12/day (Washington State Parks)
- **Northwest Forest Pass:** $30/year or $5/day (National Forest trailheads)
- **America the Beautiful Pass:** $80/year (all federal lands)
- **Mount Rainier National Park:** $35 per vehicle (7 days)
- **Alpine Lakes Wilderness Permit:** Free, required May 15 - Oct 31 for overnight stays

## Sources

Trail data sourced from [Washington Trails Association](https://www.wta.org) and [AllTrails](https://www.alltrails.com).
